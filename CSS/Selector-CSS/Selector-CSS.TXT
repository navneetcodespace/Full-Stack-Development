1. Intro to CSS
🔹 What is CSS?

CSS = Cascading Style Sheets
It is used to style HTML elements (colors, layout, spacing, etc.)

🔹 Ways to Add CSS

1. Inline CSS
<p style="color: red;">Hello</p>

2. Internal CSS
<head>
  <style>
    p {
      color: blue;
    }
  </style>
</head>

3. External CSS ✅ (Best Practice)
<link rel="stylesheet" href="style.css">
p {
  color: green;
}


🔹 CSS Syntax
selector {
  property: value;
}

Example:

h1 {
  color: red;
}


🎯 2. CSS Selectors

Selectors are used to target HTML elements

🔹 Basic Selectors

1. Element Selector
p {
  color: blue;
}


2. Class Selector
.box {
  background: yellow;
}
<div class="box"></div>


3. ID Selector
#header {
  color: red;
}



🔹 Advanced Selectors


4. Group Selector
h1, p {
  color: green;
}


5. Descendant Selector
div p {
  color: blue;
}


6. Child Selector
div > p {
  color: red;
}


7. Hover Selector
button:hover {
  background: black;
  color: white;
}
