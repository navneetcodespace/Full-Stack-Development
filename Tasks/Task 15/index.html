Objective:

The objective of this assignment is to create JavaScript functions that receive an array as argument and perform various operations on it, including finding the maximum number, calculating the sum of all elements, and counting the number of odd numbers.

Tasks :

  • Find the maximum number in the array.
  • Calculate the sum of all elements in the array.
  • Count the number of odd numbers in the array.
  • Use a dummy array for testing purposes.
  • Display the results of each operation to the user on Console.log.
